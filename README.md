# Shield-Breaker
This Project For The Minecraft Shield Breaker
Axe → Sword Macro 
Ne İşe Yarar?

Bu macro, R tuşuna bastığında:

Baltaya geçer

Sol tık atar

Kısa gecikmeyle kılıca geçer

PvP’de axe hit + sword follow-up için yapılmıştır.

Gereksinimler

Python 3.x

Gerekli kütüphaneler:

pip install pyautogui pynput

Tuş Ayarları (DEĞİŞTİR)

Aşağıdaki satırları kendi hotbarına göre ayarla:

pyautogui.press('1')  # Balta tuşu
pyautogui.press('3')  # Kılıç tuşu


1 → Balta slotu

3 → Kılıç slotu

Gecikme süresi:

time.sleep(0.06)


➡️ Daha hızlı/yavaş için değeri değiştir.
